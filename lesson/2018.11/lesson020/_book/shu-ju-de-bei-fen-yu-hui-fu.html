数据的备份与恢复

数据的备份与恢复

一、导出

导出数据库里面的结构和数据

mysqldump -u root -p 数据库名 >导出文件的路径

导出数据库里面的表的结构和数据

 mysqldump -u root -p 数据库名 表名 > 导出文件的路径

导出数据库里面的结构

mysqldump -u root -p -d 数据库名  >导出文件的路径

导出数据库里面的表的结构

 mysqldump -u root -p -d 数据库名 表名> 导出文件的路径

二、导入

退出的情况

mysql -u root -p 数据库名 < 数据库文件

进入的情况

source 数据库文件
