mysql关于表的操作

mysql数据库表的操作

1、查看所有表

show tables;

2、创建表

create table 表名(
字段 字段类型,
.......
);

注意：多个字段之间使用逗号分隔

3、查看表结构

desc 表名

4、查看建表语句

show create table 表名;

show create table 表名\G 注意：\G后面没有分号

5、删除表

drop table 表名;
